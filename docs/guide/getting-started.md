# 快速开始

本文帮助你从零开始，完成 Lyrico 的首次配置并看到你的音乐库。

## 添加音乐文件夹

Lyrico 不会自动扫描整台设备，你需要手动指定要管理的音乐文件夹。

**操作流程：**

1. 从 [GitHub Releases](https://github.com/Replica0110/Lyrico/releases) 下载最新版本 APK 并安装。
2. 打开 Lyrico，进入"歌曲"页。如果还没有添加任何文件夹，页面会显示"尚未添加文件夹"。
3. 点击 **"添加文件夹"** 按钮。
4. 在系统文件选择器中，找到并选中你的音乐文件夹，点击"使用此文件夹"进行授权。
5. Lyrico 开始扫描文件夹中的音频文件。扫描进度会显示当前状态，例如"正在扫描文件夹…"、"正在读取标签 1 / 100"、"正在更新音乐库…"。

扫描完成后，歌曲就会出现在"歌曲"列表中。如果歌曲数量较多，首次扫描可能需要几分钟。

::: tip 提示
你可以在 `设置` → `扫描设置` → `文件夹管理` 中添加更多文件夹。Lyrico 只扫描你手动添加的文件夹，不会扫描设备上的其他目录。
:::

## 浏览你的音乐库

扫描完成后，底部导航栏包含三个视图：

- **歌曲**：按列表显示所有已扫描的歌曲。点击任一歌曲进入编辑页。
- **艺术家**：按艺术家分组显示。如果开启了艺术家拆分，多个艺术家会分别归类。
- **专辑**：按专辑封面网格显示。

关于排序、本地搜索、艺术家/专辑详情页的更多操作，见 [浏览音乐库](./browsing.md)。

## 编辑第一首歌

从歌曲列表中点击任意歌曲，进入 **"编辑元数据"** 页面。这是 Lyrico 的核心编辑界面。

**快速尝试：**

1. 点击一首歌进入编辑页。
2. 修改标题或艺术家字段。
3. 点击顶部的 **"保存"** 按钮。
4. 看到"保存成功"提示后，修改已写入音频文件。

更完整的编辑操作（歌词、封面、ReplayGain 等）见 [单曲编辑](./single-song.md)。

## 安装搜索源插件

Lyrico 本身不提供在线搜索能力。歌词、封面和歌曲信息搜索需要安装 **插件** 来实现。

**操作流程：**

1. 下载插件 ZIP 文件（如从 [Lyrico-Plugins](https://github.com/Replica0110/Lyrico-Plugins) 获取）。
2. 进入 `设置` → `搜索设置` → `插件管理`。
3. 点击 **"导入插件"**，选择下载的 ZIP 文件。
4. 在"发现插件包"弹窗中勾选要安装的插件，点击"安装"。
5. 回到插件管理页，打开插件的 **启用开关**。

插件启用后，在单曲编辑页顶部搜索框输入关键词即可搜索在线歌曲信息。

更多插件使用细节见 [使用插件](./plugins.md)。
