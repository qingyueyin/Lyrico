# 单曲编辑

单曲编辑页是 Lyrico 的核心界面，用于查看和修改单首歌曲的所有元数据。

## 进入编辑页

有多种方式进入编辑页：

- 在"歌曲"列表、艺术家详情、专辑详情或本地搜索结果中 **点击任意歌曲**。
- 从其他应用（如文件管理器）**打开或分享音频文件到 Lyrico**，会直接进入编辑页。

::: warning 点击不是播放
点击歌曲的默认行为是编辑标签，不是播放。要播放歌曲，请使用更多菜单中的"播放音乐"或在编辑页中点击播放按钮。
:::

## 编辑页布局

编辑页从上到下分为以下区域：

| 区域 | 功能 |
|------|------|
| 顶部操作栏 | 保存按钮、播放按钮、更多选项 |
| 搜索栏 | 输入关键词，通过插件搜索在线歌曲信息和歌词 |
| 封面区 | 显示/更换/搜索/移除封面图片 |
| 标签编辑区 | 标题、艺术家、专辑、专辑艺术家、年份、流派、音轨号、碟号、作曲、作词、注释等字段 |
| 歌词区 | 查看/导入/导出/格式化歌词、简繁转换、时间轴偏移 |
| ReplayGain 区 | 计算曲目增益和峰值 |
| 自定义标签区 | 管理自定义标签（需要先在设置中添加可见的自定义标签 key） |

如果编辑页中缺少某些字段，可能是被 **"字段显示设置"** 隐藏了。隐藏的字段不会在保存时被无意清空。见 [元数据处理](./settings/metadata.md#字段显示设置)。

## 编辑标签并保存

**操作流程：**

1. 进入编辑页，修改需要更改的字段。字段为空时可以直接输入；已有内容的字段可以编辑或清除。
2. 点击顶部工具栏的 **"保存"** 按钮。
3. 保存成功会提示"保存成功"并返回上一页；保存失败会显示失败原因。

**重要提示：**
- 搜索结果、封面选择、歌词处理和 ReplayGain 计算都只会先更新编辑页中的临时内容。
- 只有点击 **"保存"** 后，所有修改才会写入音频文件。
- 部分 Android 版本或特定目录可能弹出额外的授权请求，按提示操作即可。

## 搜索歌词和元数据

通过插件搜索在线歌曲信息，用于补全标题、艺术家、歌词等字段。

**操作流程：**

1. 进入编辑页，顶部搜索栏会自动填入当前歌曲的 **标题 + 艺术家** 作为关键词。如果歌曲没有标题和艺术家标签，则使用文件名搜索。
2. 点击搜索按钮（或按键盘回车），Lyrico 会调用已启用的插件搜索匹配的歌曲。
3. 搜索结果页按"全部"和各插件源分标签显示。
4. 点击某个搜索结果，进入结果详情页，可以查看搜索到的元数据和歌词。
5. 在结果详情页选择操作：
   - **"应用"**：将勾选的元数据和歌词一起应用到当前编辑页。
   - **"仅歌词"**：只应用歌词内容到编辑页。
6. 回到编辑页，检查内容无误后点击 **"保存"**。

::: tip 搜索结果为空？
检查 `设置` → `搜索设置` → `插件管理` 中是否安装并启用了至少一个支持歌曲搜索的插件，以及插件配置是否填写完整。详见 [使用插件](./plugins.md)。
:::

## 封面操作

点击编辑页中的封面区域或"封面选项"，可以执行以下操作：

| 操作 | 说明 |
|------|------|
| 更换封面 | 从系统图片选择器选择本地图片 |
| 从网络搜索封面 | 通过已启用的插件搜索在线封面 |
| 选择同专辑歌曲封面 | 直接使用同专辑中找到的第一张可读封面 |
| 移除封面 | 删除当前歌曲的封面 |
| 保存封面 | 将当前封面导出到系统图片目录 |
| 裁剪图片 | 裁剪当前封面后再应用 |

网络封面搜索需要已安装并启用支持封面搜索的插件。

## 歌词操作

点击"歌词选项"可以执行以下操作：

| 操作 | 说明 |
|------|------|
| 导入 | 从文本文件读取歌词，按 UTF-8 文本处理 |
| 导出 | 导出当前歌词。TTML 导出为 `.ttml`，其他导出为 `.lrc`；空歌词不会导出 |
| 简繁转换 | 将当前歌词在简体中文和繁体中文之间转换 |
| 歌词偏移 | 通过 `-500`、`-100`、`+100`、`+500` 毫秒按钮调整时间轴 |
| 格式化歌词 | 转换歌词格式、按句排序、移除空行或移除非歌词内容 |
| 查看歌词文本 | 把逐字歌词等内容转成普通文本查看，可切换显示罗马音和翻译 |

::: warning 格式化歌词的限制
格式化歌词不能凭空生成逐字时间轴。如果源歌词没有逐字时间信息，就不能转换成需要逐字时间轴的格式。
:::

歌词偏移以毫秒为单位。正值表示整体后移，负值表示整体提前。

## ReplayGain

ReplayGain（回放增益）用于统一不同歌曲的播放音量。

**操作流程：**

1. 在编辑页找到 ReplayGain 分组。
2. 点击 **"计算 ReplayGain"**，Lyrico 开始分析当前音频。
3. 计算完成后，曲目增益、曲目峰值和参考响度等字段会填入编辑页。
4. 点击 **"保存"** 将这些值写入音频文件。

计算中可随时点击按钮取消扫描。已有 ReplayGain 值的歌曲也可以重新计算覆盖。

## 更多菜单

编辑页的更多菜单包含：

- **歌曲信息**：查看详细文件信息（时长、比特率、采样率、声道、路径、文件大小等），可复制内容。
- **分享歌曲**：通过系统分享面板发送音频文件。
- **重命名文件**：修改文件名，保留原扩展名（不会修改标题标签）。
- **删除歌曲**：删除本地音频文件，不可撤销。
