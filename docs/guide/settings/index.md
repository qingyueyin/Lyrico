# 设置概览

Lyrico 的设置页面按功能分为多个分类。本页提供快速索引，帮助你找到需要的设置项。

## 设置分类

| 分类 | 包含内容 | 详细页面 |
|------|---------|---------|
| 外观 | 主题模式、莫奈取色、种子色 | [外观](./appearance.md) |
| 扫描设置 | 文件夹管理、短音频过滤 | [资料库与文件夹](../library.md) |
| 搜索设置 | 插件管理、搜索限制、文件名匹配 | [搜索设置](./search.md) |
| 歌词 | 歌词模式、罗马音、翻译 | [歌词设置](./lyrics.md) |
| 元数据处理 | 中文转换、艺术家拆分、字段显示、自定义标签、字符映射 | [元数据处理](./metadata.md) |
| 备份与恢复 | 导出/导入应用配置 | [备份与恢复](./backup.md) |
| 其他 | 任务历史、应用日志、缓存清理 | [维护](./maintenance.md) |
