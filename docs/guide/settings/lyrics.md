# 歌词设置

控制歌词搜索、格式和写入行为。

## 歌词模式

选择搜索歌词时优先获取的歌词格式：

- **逐行歌词**：传统的逐行 LRC 格式。
- **逐字歌词**：包含逐字时间轴的歌词。
- **增强型逐字歌词**：包含逐字时间轴和更多元数据的歌词。
- **TTML**：TTML 格式歌词（Timed Text Markup Language）。

## 翻译与罗马音

- **罗马音**：开启后，搜索歌词时请求罗马音内容（适合日语歌曲）。
- **翻译**：开启后，搜索歌词时请求翻译内容。
- **仅下载翻译歌词**：开启后，如果歌曲已有原文歌词，只下载翻译部分。

## 注意事项

修改歌词模式、罗马音或翻译设置后，已有的搜索结果 **不会自动刷新**。需要重新搜索歌词才会使用新配置。

这些设置也不影响已经保存到音频文件中的歌词。
