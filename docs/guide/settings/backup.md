# 备份与恢复

## 导出设置

将当前应用配置导出为 JSON 文件。包含的内容：

- 主题和外观设置
- 搜索设置（插件启用状态、排序、配置、搜索限制等）
- 歌词设置（歌词模式、罗马音、翻译）
- 艺术家拆分规则
- 字段显示设置
- 批量匹配配置

**不包含的内容：**
- 音乐文件本身
- 歌词和封面数据
- 音乐库数据库
- 已安装的插件文件

## 导入设置

从之前导出的 JSON 文件恢复应用配置。

导入后，部分与索引相关的设置（如艺术家拆分规则）可能需要刷新音乐库或重建艺术家索引才能完全体现。

::: warning
配置备份不是完整数据备份。它只包含应用设置，不能用于恢复音乐文件、歌词数据或数据库。
:::
