# 浏览音乐库

Lyrico 提供三种浏览维度——歌曲、艺术家、专辑——以及本地搜索，帮助你快速找到想要的音乐。

## 三个主视图

底部导航栏包含三个标签页：

### 歌曲

所有已扫描歌曲的列表。默认按标题排序，支持字母侧边栏快速跳转。

- 点击歌曲进入 [编辑元数据](./single-song.md)。
- 长按歌曲或左右滑动进入多选模式（用于 [批量操作](./batch.md#进入选择模式)）。

### 艺术家

按艺术家分组的列表。如果开启了艺术家拆分（`设置` → `元数据处理` → `艺术家拆分规则`），多个艺术家的歌曲会分别归类。

点击艺术家进入艺术家详情页，详情页包含 **"单曲"** 和 **"专辑"** 两个标签页，可以查看该艺术家的所有歌曲和参与的专辑。

### 专辑

按专辑封面网格展示。点击专辑进入专辑详情页，可以查看该专辑下的所有歌曲。

专辑详情页中的歌曲列表同样支持点击编辑、长按多选等操作。

## 排序

歌曲、艺术家和专辑列表都支持排序。点击列表顶部的排序按钮可以切换排序方式：

**歌曲排序选项：**
- 标题
- 艺术家
- 修改日期
- 添加日期
- 文件大小
- 时长
- 扩展名

**艺术家排序选项：**
- 名称
- 歌曲数量
- 专辑数量

**专辑排序选项：**
- 名称
- 艺术家
- 歌曲数量
- 年份

专辑网格的列数也可以在专辑页中调整。

## 本地搜索

点击导航栏中的搜索图标进入 **"本地搜索"**。

本地搜索会同时匹配 **标题、艺术家、专辑和文件名**。输入关键词后，结果按"艺术家"、"专辑"、"单曲"三个区域分段显示。

::: warning 注意
已被隐藏文件夹中的歌曲不会出现在普通列表和本地搜索结果中。如需恢复显示，前往 `设置` → `扫描设置` → `文件夹管理` 中操作。
:::

## 播放歌曲

Lyrico 是一个标签编辑工具，不是音乐播放器。点击歌曲默认进入编辑页而非播放。

要播放歌曲：
- 在歌曲列表中，点击歌曲右侧的 **更多菜单**（⋮），选择 **"播放音乐"**。
- 在编辑页中，点击播放按钮并选择已安装的外部播放器。

Lyrico 会记住你上次选择的播放器。

## 歌曲菜单

歌曲列表右侧的更多菜单（⋮）提供以下操作：

| 操作 | 说明 |
|------|------|
| 播放音乐 | 调用外部播放器播放音频 |
| 歌曲信息 | 查看标题、艺术家、专辑、时长、比特率、采样率、声道、文件路径、文件大小等详情，可复制内容 |
| 分享歌曲 | 通过系统分享面板发送音频文件 |
| 重命名文件 | 修改文件名，保留原扩展名。**不会修改歌曲标题标签** |
| 删除歌曲 | 删除本地音频文件。**此操作不可撤销** |

在已进入多选模式时，更多菜单会被复选框替代，此时点击歌曲用于切换选中状态。
