# 常见问题

## 基本使用

### 为什么点击歌曲不放音乐？

Lyrico 是一个标签编辑工具，不是音乐播放器。点击歌曲的默认行为是进入"编辑元数据"页面。要播放音乐，请使用歌曲右侧更多菜单中的"播放音乐"。

### 保存后歌曲信息没有变化？

保存操作将修改写入音频文件本身，而不是 Lyrico 的数据库。请确认：
1. 保存时是否提示"保存成功"。
2. 文件所在目录是否有写入权限。部分 Android 版本可能需要额外授权。
3. 用其他音乐播放器查看是否显示了新标签。

### 通过其他应用或系统分享打开文件，编辑后无法保存？

从外部应用通过"分享"或"打开方式"进入 Lyrico 编辑音频时，如果出现无法保存的情况，通常是因为 Lyrico 缺少音频访问权限。

解决方法：在系统设置中授予 Lyrico 音频访问权限（设置 → 应用 → Lyrico → 权限 → 音乐和音频），授权后即可正常保存。

### 如何删除音乐库中的歌曲但不删除文件？

你可以使用"隐藏文件夹"功能（`设置` → `扫描设置` → `文件夹管理`）将歌曲从库中隐藏，文件不会被删除。

如果只是个别歌曲不想显示，可以将它们移到单独的文件夹中，然后隐藏该文件夹。

## 插件相关

### 插件安装了但搜索不到结果？

依次检查：
1. 插件是否已 **启用**（开关是否为开）。
2. 插件是否 **支持当前操作**（有些插件只支持歌曲搜索，不支持歌词或封面）。
3. 插件是否需要 **Token 或 API Key**，如果需要在插件配置中填写。
4. 网络是否正常。
5. 查看 `设置` → `其他` → `应用日志`，筛选"插件"类型查看错误信息。

### 插件 ZIP 导入失败？

常见原因：
- ZIP 包内没有有效的 `manifest.json` 文件。
- 插件 API 版本与 Lyrico 不兼容。
- 插件包结构或入口脚本不符合要求。
- 插件包超过大小限制（单个插件 5 MB，压缩包解压后 30 MB）。

具体原因会在"发现插件包"弹窗的"无法安装"区域显示。

### 如何获取更多插件？

官方插件仓库：[Lyrico-Plugins](https://github.com/Replica0110/Lyrico-Plugins)，覆盖 QQ 音乐、网易云音乐、酷狗音乐、Apple Music 等平台。

## 批量操作

### 如何一次选中多首歌？

进入多选模式有几种方式：
- **长按**任意歌曲。
- 在歌曲上**向左或向右滑动**。
- 滑动第一首歌曲设置区间起点，再滑动另一首完成区间选择。

详见 [批量操作](./batch.md#进入选择模式)。

### 批量匹配处理到一半失败了？

可能原因：
- 并发数设置过高，被对方平台限流。降低并发数后重试。
- 某些歌曲的标题/艺术家为空，搜索关键词无效。
- 插件网络请求超时。

你可以在 `设置` → `其他` → `任务历史` 中查看每首歌的具体失败原因。

### 批量重命名后文件名没有变化？

检查：
1. 占位符对应的标签是否为空（如 `@1` 对应标题，如果标题为空则生成空文本）。
2. 生成的文件名是否在预览中正确显示。
3. 是否有多首歌曲生成了相同文件名（Lyrico 会自动追加数字后缀避免冲突）。

## 歌词

### 逐字歌词为什么不显示？

- 确保歌词模式设置为"逐字歌词"或"增强型逐字歌词"。
- 插件返回的歌词需要包含逐字时间信息。如果源数据只有普通 LRC，无法转换为逐字歌词。

### 如何把繁体歌词转为简体？

在编辑页的歌词选项中，使用"简繁转换"功能。也可以在 `设置` → `元数据处理` 中设置"中文文本转换"为"繁体转简体"，此后搜索的歌词会自动转换。

## 其他

### 应用占用空间太大？

主要是封面图片缓存。前往 `设置` → `其他` → `清理缓存` 清理图片缓存。

### 更新版本后设置会丢失吗？

正常情况下不会。但建议在更新前通过 `设置` → `备份与恢复` → `导出设置` 备份配置，以防万一。
