# 资料库与文件夹

管理 Lyrico 扫描的音乐文件夹，以及音乐库的维护操作。

## 文件夹管理

进入路径：`设置` → `扫描设置` → `文件夹管理`。

文件夹管理页包含"文件夹"和"歌曲"两个标签页，分别用于查看文件夹树结构和该范围下的歌曲列表。

### 添加文件夹

点击添加按钮，在系统目录选择器中选中存放音乐的文件夹并授权。添加后，Lyrico 会自动开始扫描该文件夹。

如果已开启 `设置` → `扫描设置` → `不扫描 60 秒以下音频`，时长不超过 60 秒的音频会被跳过，适合过滤提示音、短片段和采样文件。

### 刷新文件夹

"刷新文件夹"会重新扫描对应文件夹。适合以下场景：
- 在系统文件管理器中新增了音频文件
- 删除了部分音频文件
- 用其他工具修改了文件的标签

### 隐藏与显示文件夹

**隐藏文件夹**：隐藏该文件夹及其子文件夹中的歌曲。隐藏后歌曲不出现在歌曲列表、艺术家列表、专辑列表和本地搜索结果中。

**显示文件夹**：恢复显示之前隐藏的文件夹及其歌曲。

隐藏不会删除文件，也不会移除文件夹记录。适合临时排除某些文件夹（如铃声、播客）的场景。

### 移除文件夹

从 Lyrico 中移除此文件夹，并清除音乐库里的对应歌曲记录。本地音频文件不会被删除。之后仍可重新添加此文件夹。

### 操作对比

| 操作 | 本地文件 | 文件夹记录 | 歌曲是否显示 |
|------|----------|------------|-------------|
| 隐藏文件夹 | 保留 | 保留 | 不显示，可再次显示 |
| 移除文件夹 | 保留 | 移除 | 不显示，需重新添加 |

## 扫描设置

`设置` → `扫描设置` 中包含以下选项：

- **文件夹管理**：管理已添加的音乐文件夹，入口同上。
- **不扫描 60 秒以下音频**：开启后跳过时长 ≤ 60 秒的音频文件。

## 文件夹权限问题

如果出现以下情况，Lyrico 会自动清理对应的音乐库记录并刷新索引：
- 文件夹权限丢失（如系统回收了存储权限）
- 文件夹被删除或移动
- 数据库中的歌曲在设备上已不存在

如果遇到"部分文件无法访问"的提示，可以尝试：
1. 进入 `文件夹管理`，移除受影响的文件夹。
2. 重新添加该文件夹并授权。
